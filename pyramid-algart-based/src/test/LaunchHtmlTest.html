Please create folder /pp-links/55e9afimageio/ and place there config.json with description of the pyramid, for example:

    {
    "pyramidPath": "C:/Image/BIG/ImageIO/withPPJson",
    "renderer": {
        "format": "jpeg"
    }
}

withPPJson folder must contain 2 files:

  • data.tif - some image
  • .pp.json - configuration file with the following minimal content: 
        {
        "fileName": "data.tif",
        "formatName": "imageIO"
    }

Then start LaunchJavaTest.cmd. Note that the following JARs must present in the subfolder .m2:

algart-1.2.7.jar
grizzly-framework-2.3.22.jar
grizzly-http-2.3.22.jar
grizzly-http-server-2.3.22.jar
grizzly-http-servlet-2.3.22.jar
jai-imageio-core-1.3.1.jar
javax.json-1.0.4.jar
javax.json-api-1.0.jar
pyramid-algart-based-1.0.1.jar
pyramid-common-1.0.1.jar
pyramid-http-1.0.1.jar
simagis-plane-pyramid-sdk-1.2.13.jar

and pyramid-algart-based/target subfolder must contain normally compiled class-files.

Please copy subfolders

    pp-links
    pp-images

into the root of your disk.

Then start

    LaunchJavaTestGnu.cmd

Note: you need to have the following additional JARs from GNU library in .m2 subfolder:

    loci_tools-5.0.6.jar
    PlanePyramidSourceBioFormats.jar

You may use LaunchJavaTest.cmd to test without additional GNU JARs.

Some ready test for simple image (gif)

To add new image, please use

    AddImage.cmd someImage.jpg

for standard formats (jpg, png, gif) or

    AddImage.cmd someImage.svs -gnu

for pyramid formats supported by LOCI. The links to added images will appear, correspondingly, in the files

    PyramidsStandard.html and
    PyramidsGnu.html

Warning! Please use the same disk drive for /pp-links, /pp-images folders and added images. AddImage.cmd adds links to /pp-links and /pp-images at the current drive.

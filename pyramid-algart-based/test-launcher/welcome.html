Please copy demo files to the correct place on your disk by the following cmd-file:

    CopyDemoFiles.cmd

This cmd-file will copy the subfolders pp-links and pp-images into the root of your disk. If it will fail due to insufficient permissions, please copy these folders into the root of the current disk manually.

Then please start the service by the following cmd-file:

    LaunchJavaServers.cmd

Note:

  1. This cmd-file starts Java Machine via call

        java -Xmx8g -classpath %CP% net.algart.pyramid.standard.tests.OpenSourceImageFileAccessTest -gnu

    It means that java.exe must be available in system paths, and it must be 64-bit (not 32-bit). If this call fails, please replace "java" by the full path to the last installed 64-bit JRE, version 8 or higher, for example:

        "C:\Program Files\Java\jre1.8.0_91\bin\java.exe" -Xmx8g -classpath %CP% net.algart.pyramid.standard.tests.OpenSourceImageFileAccessTest -gnu
  2. This cmd-file should be started only once. Attempt to start this file 2nd time will fail with an error message "java.net.BindException: Address already in use: bind". If you want to stop previously started service, please switch to the console window when it is started (usually with white text on black background) and click Ctrl+C.

  3. Also note: you need to have the following additional JARs from GNU library in pp-config/lib/maven subfolder: 

        loci_tools-5.0.6.jar
    PlanePyramidSourceBioFormats.jar

    Usually this demo is distrubuted with these JARs. But if not, the cmd-file will start anyway, but only standard formats (jpg, png, gif) will be available for viewing.

To add new image, please use

    AddImage.cmd someImage.jpg username

The user name can be any simple word, like "Daniel", "Anil", etc — it is just the name of subfolder in /pp-image folder where the images are added. The links to added images will appear at the end in the file

    pyramids.html

Please open this file! Initially, it can contain links to some already existing demo images.

Or please open ready multifile demo page with a list of some built-in pyramids.

Warning! Please use the same disk drive for /pp-links, /pp-images folders and added images. AddImage.cmd adds links to /pp-links and /pp-images at the current drive.
